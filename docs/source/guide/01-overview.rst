Overview
========

Ox-fin is an accounting and reporting engine backed by Django. It does not provide a user interface,
but command line tools utilities on the top of a library that can be reused.

Features:

- Create, edit and manage ledger books, journals, accounts, entries;
- Reporting system, with complex formula and integrated DSL for data selection;
- Create, update and edit template for ledger books and reports;
- Import ledger book and report templates from YAML
- Import ledger book from XLSX files;

More details on the project's README.
